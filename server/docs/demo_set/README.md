LexiPro Demo Exhibit Set

This folder contains a 4-document demo set designed for high-stakes forensic validation.

1) Anchor_Agreement.pdf
   Multi-page agreement used for grounding, clause pinpointing, and Bates numbering.

2) Email_Thread.pdf
   Metadata-rich email chain used for custody extraction and PII/privilege shielding.

3) Financial_Statement.pdf
   One-page target used for sabotage challenge and heartbeat detection.

4) Contradictory_Memo.pdf
   Short witness statement that contradicts the agreement or email trail for conflict analysis.
