# LexiPro Strategic Roadmap

## Current Engine (v4.8.2-commercial)
- **Multi-Tenant Architecture**: Hard-scoped PostgreSQL persistence.
- **Evidence Anchoring**: PDF text-layer extraction to page/line/bbox.
- **Tamper-Evident Logs**: Cryptographic hash chaining on all forensic events.
- **Zero-Retention Proxy**: Server-side LLM calls with forensic JSON enforcement.
- **IP Integrity**: Verified dependency inventory and MIT commercial licensing.

## Immediate Sprint (Lexis/CaseMap Baseline)
- **4-Task Home Shell**: Ask, Draft, Summarize, Upload-QA task tiles with prompt rails.
- **Citation-First UX**: No-anchor/no-output enforcement with click-to-teleport evidence.
- **Search Term Hit Map**: PDF viewer hit map with click-to-jump anchors.
- **Trust Center**: Retention controls, purge action, encryption/no-train posture panel.
- **Research Accelerators**: Saved filters, folders, history map, alerts, search-within-results.
- **Evidence Drafting Mode**: Playbook hooks, deviation detection, and exportable authority packet.

## Phase 2 (Q2 2025)
- **Advanced OCR**: Multi-column handling for scanned medical records using Gemini 2.5 Image models.
- **Real-time Video Anchoring**: Frame-level timestamp citations for bodycam/surveillance footage.
- **SAML/OIDC Integration**: Support for large firm identity providers.
- **Direct Clio/Everlaw API**: Two-way document synchronization.

## Phase 3 (Q3 2025)
- **Predictive Legal Research**: Grounding LLM responses in real-time jurisdiction-specific case law.
- **Collaborative Redaction**: Real-time multi-user PII masking with audit verification.
