# RAGFlow DeepDoc - Summary

Source: https://github.com/infiniflow/ragflow/tree/main/deepdoc

What it is:
- A document parsing and vision toolkit from RAGFlow with OCR, layout recognition, and table structure recognition (TSR).
- Supports PDF parsing with bounding boxes and table/figure extraction.

Key capabilities:
- OCR for images/PDFs with position output.
- Layout detection (text/title/figure/table/captions/header/footer/reference/equation).
- Table structure recognition with cell types.
- Auto-rotation for tables in scanned PDFs.
- Parsers for PDF/DOCX/Excel/PPT.

Why it matters to CaseCompanion:
- Can improve evidence ingestion by extracting structured tables and layout-aware text.
- Useful for medical bills, police reports, and forms where tables matter.
