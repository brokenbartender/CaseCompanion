# OpenClaw (openclaw/openclaw) - Summary

Source: https://github.com/openclaw/openclaw

High-level description:
- Local-first AI assistant platform with a gateway control plane and multi-channel messaging support.
- Provides agent routing, tools (browser/canvas/nodes), and a skills system for extensions.

Potential app learnings (inspiration only):
- Modular agent/session architecture for separating case workspaces.
- Skills-style extension system for adding procedural wizards.
- Control UI pattern for system status + diagnostics.
