# MCL 750.81 - Assault and Battery (Michigan)

Source: https://www.legislature.mi.gov/documents/mcl/pdf/MCL-750-81.pdf

What it is:
- Michigan Penal Code section defining assault and battery and penalties.

Key points (high-level):
- Basic assault/battery is a misdemeanor (up to 93 days and/or fine).
- Enhanced penalties for certain victim categories (e.g., health professionals).
- Domestic-relationship variants and repeat-offender enhancements.
