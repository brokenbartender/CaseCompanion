# OpenContracts - Repo Summary

Source: https://github.com/Open-Source-Legal/OpenContracts

High-level description:
- OpenContracts is an AGPL-3.0 licensed platform for document analysis, annotation, and collaboration.
- It includes document processing, AI-assisted analysis, and structured data extraction.

Core capabilities called out in the README:
- Document processing with automated structure extraction.
- Annotation and analysis with custom schemas and relationships.
- AI agents for document search and analysis.
- Collaboration with threaded discussions and moderation.
- Data extraction across document collections.
- Document version control and recovery.

Architecture notes (from README):
- Parser/embedding/thumbnail pipeline, with pluggable parsers.
- Vector embeddings for semantic search.
- PDF-first layout-aware data format for annotations.

License:
- AGPL-3.0.
